# DayClock

Android Studio project in Java.

## Open
1. Open Android Studio.
2. Choose "Open".
3. Select this folder.
4. Wait for Gradle sync.
5. Run the app on a device or emulator.

## Features
- Daily tasks as time intervals
- 24-hour analog clock sectors
- Task list tab with add/edit/delete
- Color selection for tasks
- Notifications at task start time
